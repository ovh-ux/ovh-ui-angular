## Usage

### Information

```html:preview
<oui-status type="info">Status</oui-status>
```

### Success

```html:preview
<oui-status type="success">Status</oui-status>
```

### Warning

```html:preview
<oui-status type="warning">Status</oui-status>
```

### Error

```html:preview

```

| Attribute               | Type            | Binding | One-time binding | Values                                 | default | Description                            |
